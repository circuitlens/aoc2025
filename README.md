# Advent of Code 2025
My solutions for [Advent of Code 2025](https://adventofcode.com/2025/) problems, written in Python. Aside from the second half of day 10's assignment, all problems are completed. All programs require you to save your input as a text file in the same folder as them, named `day#_input.txt`. They run in the command line. Python 3.x must be installed on your device.
